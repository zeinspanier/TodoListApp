Todo Lists

This is a web app to create todo lists

Instructions are as follows:

1: Click 'Create New List' and enter a name for the list.(duplicates are not allowed)
2: Type item into the text box, use 'add' and 'remove' to edit the list.
3: To cross an item off the list click on the item. To undo cross off click the item again.
4: To delete a list perminitly, click delete.
